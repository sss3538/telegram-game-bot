# Setup and Run Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/sss3538/telegram-game-bot.git
   ```

2. Navigate to the project directory:
   ```bash
   cd telegram-game-bot
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Run the application:
   ```bash
   npm start
   ```

5. Configure any necessary environment variables as needed before running.
